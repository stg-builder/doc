Script commands

Label group

IN

Jump to OUT with the same label number.
  • Label number
    You can set any value you like
    If you have more than one label IN with the same number, it doesn't matter.

OUT

The label number jumps from the same IN
  • Label number
    You can set any value
    Only one label OUT of the same number can be placed
You can't jump to another task
