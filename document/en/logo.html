Shooting Game Builder LOGO

Usage Guidelines

The logo may be used without permission only in the following cases


  • Creating a game using the Shooting Game Builder and displaying it in that game
  • Create a game using the Shooting Game Builder and display it when sending out information about it
  • Displayed when sending out information about the Shooting Game Builder
    •
    Logo type and size

    There are two types of different layouts
    It can be resized between minimum and maximum size

    Type A 160x80 - 800x400
    Type B 200x100 - 600x300

    Prohibited matters


  • Modification of the logo (design, coloring, resizing out of range, deformation, etc.)
  • Displaying the logo in places that are not related to Shooting Game Builder in any way
  • Redistribution of the image itself
