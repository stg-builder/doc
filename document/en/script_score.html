Script commands

Score Group

Set

Rewrites the score set for the character
  • Target to be set up
  • Renew the score
  • Child ID
    ID for a child
  • You can specify a configuration score
    value or a variable

Add

Add the score directly
  • Additional score
    You can specify the score to be added as a value or variable

Conditions

Determines if a character has multiple children and if they were all destroyed in time
  • Timer
    Specifies a time limit in frames
  • Start the timer after one of the children is destroyed
    If you don't check, the timer will start immediately
  • Child ID
    Up to six children can be set to be destroyed
