Editing the formation

Combining multiple characters into a single formation reduces the complexity of appearance management
You can edit the formation you created here with Edit Stage by adding You can make it appear instead of a character

Here's help for common operation of the edit window

  • Script
    character This takes precedence over scripts registered in Optional
  • XY offset
    Set the position of each character in relative coordinates
  • Delay
    Delay
    This sets the time between the appearance of the character and the appearance of the next character in frames
  • Annihilation bonus points
    Points added for destroying the entire formation
  • Synchronize appearance coordinates with background on appearance delay
    You might want to check for ground objects that synchronize with the background
  • Script (Overwrite)
    This will be overwritten by replacing the script set in the character edit
  • Variable 15
    When it appears as a formation, variable 15 is set to a value of +1 from 0
