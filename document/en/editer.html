Editor settings

  • Save the project before debugging
    Automatically overwrite it
  • Background Color
    This will be the background color of the image preview in sprite editing and background editing
  • Initialize edit window pos.
    If the edit window goes off-screen
  • Initialize all category settings
    Checks and names of category settings will be initialized
  • Autosave
    Saves the project as an alias at a fixed time. Specify interval and number of saves
  • Place category/search to the left
    Change the arrangement of the main window
  • No SBD file version check
    Eliminate the "This version of the SBD not be read" message
  • Export
    Writes all part data on disk
    Drag & drop import into your project
    In this case, the link information between parts will be destroyed
