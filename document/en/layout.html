Edit Layout

Here's help for common operation of the edit window
  • Layout parts
    Select the parts you want to place from here and drag and drop them to the center
  • It's a list of parts that are already in place. The detailed information of the selected part will be displayed at the bottom and you can edit it
  • Preview
    It will show you the position and size of the part you are placing
If you don't specify a font, you can use to specify the font. Display in the default font
  • Display the text
    with the specified font
  • Main Area
    This is the area where the actual play content is displayed. Must have at least one
  • Sprite
    Show the specified sprite
    • Specify a pattern and animation number
  • Scores
    Displays the score in the specified font.
  • Remaining machine
    Display the number of remaining machines in a specified font
  • Bomb
    Display the number of bombs remaining in the specified font
  • Gage
    Draw a gauge for the player or a boss HP gauge. Draw gauges using patterns and animation numbers with registered sprites
  • Ranking
    Parts for displaying a list of rankings in a ranking scene. Edit the content in the advanced settings
  • Weapon icon
    Displays the player's weapon icon. Edit the content in the advanced settings
  • Powerup Panel
    Show the powerup panel for the player. You will also need the Power Up Panel in the player edit
  • Number of combos
    Display the number of combos in the specified font
  • Display the contents of the variable
    in the specified font
