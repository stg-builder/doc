Script commands

physics group

  • Gravity
    Change the gravity setting in the game settings
  • Parameter
    Change physics settings in character editing
  • Force
    Gives movement force to the specified direction
    If "continue" is checked, it will continue to give movement force after that
  • Impuls
    Applies acceleration in the specified direction
  • Torque
    Applies rotational force
Force,Impuls,Torque,Variables can be specified, but only integers can be handled.
