Distribution builds

Output the data and program for distribution

After selecting the destination folder, click Start Build
    The following files are required for execution. All will be copied to the output folder
  • GAME.DAT
    Data file
    If an output file name is specified in the game settings, it will be that file name
  • GAME.BMP
    Splash image
    If an output file name is specified in the game settings, it will be that file name
  • SB_CORE.EXE
    Executable file
    If an output file name is specified in the game settings, it will be that file name
