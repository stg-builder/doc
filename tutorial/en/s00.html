The sample projects are stored in the samples folder under the folder where Shooting Game Builder is installed.
Image and audio resources needed for each project are located in the resources folder.

If you have edited and saved the content, the saved content will be lost when the application is updated
Please copy the entire samples folder to another location and edit it before making any modifications.
